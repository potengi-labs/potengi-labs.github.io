# potengi-labs.github.io
Ecossistema Potengi Labs

Uma Joint Venture Operacional voltada ao desenvolvimento, implantação e validação de soluções tecnológicas para gestão integrada do Ecossistema Prédio Vital, utilizando metodologia ágil, inteligência operacional e desenvolvimento progressivo de sistemas web.

O projeto terá como objetivo inicial a criação de um sistema piloto para gestão do Condomínio Vital, servindo como modelo para replicação em outras unidades de negócio e vertentes operacionais do ecossistema.
